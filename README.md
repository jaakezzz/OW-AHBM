# OW-AHBM
# Overwatch Workshop: • ANY HERO BOSS MODE •

An ongoing, passion project since pre-workshop era for a custom Overwatch game mode. This project originally began as a hobby during my university years and has evolved into a massive, feature-rich script.

## ⚠️ Project Status: Refactoring
The script currently sits at **~20,000 lines of code** and is starting to hit the Overwatch Workshop's element limits. 

Because this was built progressively over nearly a decade, the codebase is heavily unorganized and contains legacy logic. **The primary goal of this GitHub repository is optimization and refactoring.**

## 🤝 Contributing
If you are a fellow Workshop developer looking to help optimize, clean up legacy logic, or suggest modular fixes, contributions are incredibly welcome! 

To keep the code stable, please fork the repo and follow these steps:

* **Branch from Staging:** Please ensure your local feature branch originates from the up-to-date `staging` branch.
* **Target Staging for PRs:** When your code is ready for review, submit your Pull Request aimed at the `staging` branch (direct pushes to `main` and `staging` are blocked).
* **Highlight Changes:** Use the GitHub diff tool or PR description to highlight exactly which rules/features you've optimized or removed.
